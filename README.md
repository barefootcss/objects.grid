#Grid Object

The `grid` object arranges content using a grid system.

	<div class="grid">
		<div class="grid__item w-1-3"></div>
		<div class="grid__item w-1-3"></div>
		<div class="grid__item w-1-3"></div>
	</div>

## Installation

Install using Bower:

	$ bower --save bare-grid

Import into project:

	@import "bower_components/bare-grid/objects.grid";
